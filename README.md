 📊 Customer Behaviour Analysis


📌 Project Overview
This project focuses on analyzing customer data to understand their shopping behavior, preferences, and purchasing patterns. The goal is to extract meaningful insights that can help businesses make better decisions.

🔍 Work Done

In this project, the dataset was cleaned by removing errors and correcting category mappings to ensure accuracy. Missing (null) values were identified and treated properly based on the data. Exploratory Data Analysis (EDA) was performed to understand patterns and trends. Customer reviews and ratings were analyzed to measure satisfaction, and previous purchase data was studied to understand buying behavior. Duplicate records were checked and handled, and column headers were renamed to make them simple and easy to understand.
